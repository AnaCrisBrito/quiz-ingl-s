Sobre o Quiz

Este quiz foi desenvolvido para ajudar estudantes iniciantes de inglês a praticarem o uso correto do verbo to be — um dos verbos mais importantes da língua inglesa.

Você encontrará 10 perguntas com três opções de resposta: am, is e are. O objetivo é escolher a alternativa correta para completar a frase de forma gramaticalmente correta.

Este projeto foi criado com HTML, CSS e JavaScript e faz parte de um exercício de aprendizagem prática. Além de reforçar o conteúdo gramatical, o quiz também desenvolve habilidades em desenvolvimento web.

Sinta-se à vontade para testar seus conhecimentos e repetir o quiz quantas vezes quiser!
